# Sesami 설계 문서 v2

이 디렉터리는 `docs/design`(v1)과 `projectreadme.md` 사이에서 확인된 격차를 해소하기 위해 새로 정리한 프로젝트 계획서를 담고 있습니다. Graph-RAG 도입, 멀티 에이전트 오케스트레이션, AWS 마이그레이션 세부 사항과 같은 최신 의사결정이 필요할 때는 항상 v2 문서를 참고하세요.

## 주요 변경 사항
- **Graph-RAG & 시맨틱 검색**: v1에서는 지식 그래프 로드맵이 부족했습니다. v2는 스키마, 적재 파이프라인(Tree-sitter + Neo4j), 워커가 노출해야 할 조회 규약까지 기술했습니다.
- **에이전트 워크플로 정의**: Step Functions 팬아웃, AWS Batch 잡 스펙, 실패 허용치가 명시되어 `worker/run_analysis.py` 구현을 막고 있던 공백을 해소했습니다.
- **전달 계획 갱신**: 누락되었던 그래프 적재, 시맨틱 인덱싱, 관측성 작업을 포함하도록 12주 일정(기존 06_IMPLEMENTATION_PLAN.md 기반)을 재정렬했습니다.
- **운영 가드레일**: 시크릿 관리, 인프라 경계 조건, PR 검증 게이트에 대한 런북을 추가했습니다.
